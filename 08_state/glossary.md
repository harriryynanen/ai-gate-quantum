# Glossary

## Analysis session
Yksi käyttäjän analyysikeskustelu + siihen liittyvät datasetit, mappingit, jobit ja tulokset.

## Solver
Versionhallittu Python-paketti, joka ratkaisee tietyn ongelmaluokan.

## Solver registry
Lista solveista, joita järjestelmä virallisesti tukee.

## Canonical schema
Solverin odottama vakiorakenne input-datalle.

## Mapping
Muunnos käyttäjän alkuperäisestä datasta canonical schemaan.

## Result schema
Rakenteellinen määrittely solverin tuloksille.

## Result renderer
UI-kerros, joka renderöi tulokset result scheman perusteella.

## Run-scoped AI panel
Tulossivun AI-paneeli, jolla on konteksti juuri kyseisestä ajosta.

## Quantum credibility
Vaatimus, että kvanttiosuus näyttää oikeita Qiskit-artefakteja, ei vain kvanttiaiheista ulkoasua.
